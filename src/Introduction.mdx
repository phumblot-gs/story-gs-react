# UI Component Library

This is a modern, responsive UI component library built with React, Tailwind CSS, and TypeScript, focusing on accessible and customizable UI components with consistent design language.

## Installation

```bash
npm install gs-components-library
```

## Usage

```jsx
import { Button, ButtonSmall, ButtonCircle, ButtonStatus, StatusIndicator } from 'gs-components-library';
import 'gs-components-library/dist/style.css';

function App() {
  return (
    <div className="space-y-4">
      <Button>Regular Button</Button>
      <ButtonSmall>Small Button</ButtonSmall>
      <ButtonCircle icon="Plus" />
      <ButtonCircle>A</ButtonCircle>
      <ButtonStatus status="FOR_APPROVAL" icon="Check" />
      <StatusIndicator status="VALIDATED" size="medium" />
    </div>
  );
}
```

## Components

This library includes the following components:

### Button Components

- **Button**: Standard button component for primary actions
- **ButtonSmall**: Compact button for secondary or tertiary actions
- **ButtonCircle**: Circular button with icon or letter
- **ButtonStatus**: Status-specific button with appropriate colors for approval workflows

### Status Components

- **StatusIndicator**: Visual indicator for different status states

## Features

- **Fully typed components** with TypeScript support
- **Accessible components** built on Radix UI primitives
- **Responsive design** with mobile-first approach
- **Consistent styling** using Tailwind CSS
- **Flexible theming** with customizable properties
- **Icon system** integrated with Lucide React

## Component Properties

### Button & ButtonSmall

```jsx
<Button 
  background="white" // 'white' | 'black' | 'grey'
  indicator={false} // boolean - shows indicator dot
  disabled={false} // boolean
  featured={false} // boolean - gives more prominence
>
  Button Text
</Button>
```

### ButtonCircle

```jsx
<ButtonCircle
  background="white" // 'white' | 'black' | 'grey'
  disabled={false} // boolean
  featured={false} // boolean - gives more prominence
  icon="Plus" // Use any of the allowed pictograms
  indicator={false} // boolean - shows indicator dot
  size="large" // 'small' | 'large'
>
  A {/* Single letter (optional, not needed when using icon) */}
</ButtonCircle>
```

### ButtonStatus

```jsx
<ButtonStatus
  disabled={false} // boolean
  icon="Check" // 'Check' | 'X'
  isActive={false} // boolean
  size="large" // 'small' | 'large'
  status={MediaStatus.FOR_APPROVAL} // MediaStatus enum
/>
```

### StatusIndicator

```jsx
<StatusIndicator
  size="medium" // 'small' | 'medium' | 'large'
  status={MediaStatus.VALIDATED} // MediaStatus enum
/>
```

## Requirements

- React 18+
- Tailwind CSS 3+
