# UI Component Library

This is a modern, responsive UI component library built with React, Tailwind CSS, and TypeScript.

## Installation

```bash
npm install gs-components-library
```

## Usage

```jsx
import { Button, ButtonSmall, ButtonCircle, ButtonStatus } from 'gs-components-library';
import 'gs-components-library/dist/style.css';

function App() {
  return (
    <div>
      <Button>Regular Button</Button>
      <ButtonCircle icon="Plus" />
      <ButtonSmall>Small Button</ButtonSmall>
      <ButtonStatus status="FOR_APPROVAL" icon="Check" />
    </div>
  );
}
```

## Components

This library includes the following components:

- **Button**: Standard button component with various styling options
- **ButtonCircle**: Circular button with icon or letter
- **ButtonSmall**: Small text button
- **ButtonStatus**: Status-specific button with appropriate colors
- **StatusIndicator**: Simple indicator for different status states

## Features

- Fully typed TypeScript components
- Tailwind CSS styling
- Responsive design
- Accessible components
- Customizable themes
- Consistent design language

## Requirements

- React 18+
- Tailwind CSS 3+
